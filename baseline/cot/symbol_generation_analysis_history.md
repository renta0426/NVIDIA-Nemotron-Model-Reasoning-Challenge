### 2.2 result/2_2_1

- symbol は 25/60 で維持。numeric_2x2 が 24/40、glyph_len5 が 1/20、symbolic answer type は 2/24。

## 3. 横断的な所見

- verified_trace_ready の numeric_2x2 はどの result でもほぼ安定している。symbol の変動は主に answer_only_keep numeric と manual glyph の扱いで説明できる。
